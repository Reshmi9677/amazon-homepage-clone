# amazon-homepage-clone

A responsive, pixel-perfect frontend replica of the Amazon homepage built to master semantic layout structure and modern responsive CSS.

##  Features

*   **Header Navigation**: Authentic layout containing the logo, location pin, search bar, and cart tally counter.
*   **Sub-Nav Banner**: Category listings with an interactive "All" dropdown menu layout.
*   **Hero Carousel**: Fluid background image banner showcasing rotating retail deals.
*   **Product Grid**: Dynamic, multi-column card layout that scales responsively across viewport breakpoints.
*   **Footer Links**: Organised site directory links complete with a "Back to Top" scrolling button.


##  Tech Stack

*   **HTML5**: Semantic tags (`<nav>`, `<header>`, `<main>`, `<footer>`)
*   **CSS3**: Flexbox alignment, CSS Grid positioning, and custom variables
*   **Icons**: FontAwesome / Boxicons vector assets
